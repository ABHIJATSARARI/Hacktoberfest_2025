# Hangman Game
## Game Details
**Language**: Python
**Description**: Hangman is a classic word-guessing game where the computer randomly selects a secret word, and the player tries to guess it one letter at a time.

**Files Created:**
- hangman.py
- README.md

## How to Run
1. Make sure Python 3.x is installed.
2.  Navigate to the game's directory from your terminal:
    ```sh
    cd python/hangman
    ```
3.  Run the script using Python:
    ```sh
    python hangman.py
    ```
